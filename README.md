# JIRA-TICKET-CLONE
It is a Project Managing tool which is very helpful in tracking the issues and fixing the bugs in software development industry 

# How to use it
1. Click on the plus sign to generate a ticket and add some task to it then enter "shift" it will generate a ticket for you.
    
    <img width="854" alt="image" src="https://user-images.githubusercontent.com/62813226/144700562-bb98cb01-1a81-4ff3-8cea-c06280e70bd2.png">

    <img width="851" alt="image" src="https://user-images.githubusercontent.com/62813226/144700514-1630aa7a-f50c-42b4-bc67-af115be29079.png">
3. You can change the color or priority of the tikcet by just clicking the color on ticket.
    
    <img width="854" alt="image" src="https://user-images.githubusercontent.com/62813226/144700743-ed17067b-d3ba-4cf8-bd78-0e0681192736.png">

5. You can edit the task by clicking on the lock icon and after you have the changes unlock it.

   <img width="853" alt="image" src="https://user-images.githubusercontent.com/62813226/144700684-342dcc94-c49c-4eee-b8d9-79969bd72c63.png">

6. You can fiter the tickets by clicking on the priority color toolbox. 

   <img width="854" alt="image" src="https://user-images.githubusercontent.com/62813226/144700708-7f1b25dd-c36d-4640-be41-baa5584cc14e.png">

8. By double clicking you will get all the tickets.
    
   <img width="853" alt="image" src="https://user-images.githubusercontent.com/62813226/144700639-9917719e-fdee-49e9-887e-6fc6bf7e8d51.png">

8. You can delete the ticket by clicking on cross icon and then simulataneosly clicking on any of the ticket after you have deleted, you can click again to undo the effect of  cross icon.
